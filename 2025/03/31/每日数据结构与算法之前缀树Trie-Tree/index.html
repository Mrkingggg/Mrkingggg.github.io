前缀树 Trie Tree

1. 前缀树

前缀树是一种N叉树，每一个节点代表一个字符串或前缀

注意：根节点为空字符串

2. 子节点如何表示

1. 数组 SubNode[26] nodes: targetChar - 'a'为下标

查询时看当前节点的子节点数组是否有相应字符 SubNode[thisChar - 'a']是否为空

插入时先查询，如果有则进入该节点；若无则插入

2. HashMap： key = 字符， value = SubNode

在只考虑26个字母的情况下，没有比数组查询慢很多，但不用预先创建长度为26TreeNode的数组

3. Insertion

1. 创建空字符串为初始根Node
2. 判断

​ 如果存在当前遍历到字符的子节点 –> 当前节点 = 子节点；

​ 反之新建节点，并存入子节点，最后当前节点也要变为子节点

3. boolean标记： 如果某个词不是前缀树leaf但已经结束，则需要标记该节点为isWord
