LC912 912. 排序数组 - 力扣（LeetCode）

插入排序

要素：

  • pivot 从 0 到 length - 1

  • 每次插入的数为 nums[pivot + 1]

  • 查找插入位置并移动数组

​ 时间复杂度O(n^2)， 空间复杂度O(1)

快速排序

要素：双指针对向法, 这一次先分左右大小，找出下一次边界，再往小递归

  • pivot = (int)(Math.random() * (r - l + 1) + l) 随机基准

  • swap(nums, left, pivot) 将基准放在局部数组最左nums[left]

  • 交换比nums[left]小的右侧数和比nums[left]大的左侧数, 交换条件：start <= end

  • 局部交换结束（ left > right) –> 交换nums[left]和nums[end], 返回交换结束位置end

  • 整体递归左右局部交换

归并排序

要素：

  1. 先递归左[left, mid]右[mid+1, right]小区间再排序本层（合并）

  2. 辅助数组

堆排序的应用

LC215. 数组第k大元素

堆排序API – PriorityQueue

  • 添加元素：offer(ele) , add(ele)

  • 移除堆顶元素：poll()

  • 设置大根堆 / 小根堆 – Comparator

LC122 买卖股票最佳时机Ⅱ

“ 给你一个整数数组 prices ，其中 prices[i] 表示某支股票第 i 天的价格。

在每一天，你可以决定是否购买和/或出售股票。你在任何时候 最多 只能持有 一股 股票。你也可以先购买，然后在 同一天 出售。

返回 你能获得的 最大 利润 。”

通解： 递归状态机

  • dfs[i][0] 代表第i天未持有股票的利润

  • dfs[i][1]代表第i天持有股票的利润

  • 前一天持股，后一天卖掉变为[0]; 前一天非持股，后一天买入变为[1]；其他情况保持原状态

  • 每一天最大利润为前一天买/卖/不变的最大值

    dfs[i][0] = Math.max(dfs[i-1][1] + prices[i], dfs[i-1][0])

​ dfs[i][1] = Math.max(dfs[i-1][0] - prices[i], dfs[i-1][1])

  • 边界条件：

    dfs[-1][0] = 0, dfs[-1][1] = -∞

  • 递归入口：dfs[n-1][0], 因为dfs[n-1][1]卖不出去注定小于前者

    递归结果： 超时

    所以优化为dp：

核心：

  2. dp[i][0] = dp[i-1][0] + grid[i][0] // 头列赋值

LC62.不同路径

核心：

  1. dp[0][0] = 1 原点到原点一条路
  2. dp[i][0] = dp[i-1][0]
  3. dp[0][j] = dp[0][j-1]
  4. dp[i][j] = dp[i-1][j] + dp[i][j-1]

return dp[row-1][col-1]

LC63.不同路径Ⅱ

有障碍 grid[i][j] == 1时路不通

核心：

  1. if(grid[0][0]==1) dp[0][0] = 0; else dp[0][0] = 1 // 有障碍原点无路，无障碍原点路径数1
  2. if(grid[i][0]==1) dp[i][0] = 0; else dp[i][0] = dp[i-1][0]
  3. if(grid[0][j]==1) dp[0][j] = 0; else dp[0][j] = dp[0][j-1]
  4. if(grid[i][j]==1) dp[i][j] = 0; else dp[i][j] = dp[i-1][j] + dp[i][j-1]

return dp[row-1][col-1]

法1： 摩尔计数法：时间复杂度O(n), 空间复杂度O(1)

遍历数组：

  • 当前数是众数，次数加一
  • 当前数不是众数，次数减一：
    • 如果次数 <= 0， 众数变为当前数，次数重置为1

遍历结束，返回众数
